# Habit & Workout Tracker

A full-stack habit and workout tracking app built with React, Express, and MongoDB. Users can log daily habits and visualize weekly progress.

## Features
- Create, complete, and delete habits/workouts
- Weekly progress chart (data visualization)
- Responsive dashboard
- Persistent MongoDB storage

## Tech Stack
React, Express, MongoDB, Recharts
